KEY INTERACTIONS
═══════════════════════════════════════════════════════════════════════════════════

1. Dropdown Change
   User selects new provider → 
   Show loading state on cost →
   Fetch new calculation →
   Animate cost change →
   Update provider badges

2. Auto-Optimize Button
   Click button →
   Show "Calculating..." →
   Call algorithm endpoint →
   Update all dropdowns →
   Show new cost with animation

3. Provider Dropdown
   - Only show providers that offer this item
   - Disabled if only one provider available
   - Show current selection as default

4. Cost Animation
   When cost changes:
   - Smoothly count from old to new value
   - Use 300ms transition
   - Flash green/red based on delta
